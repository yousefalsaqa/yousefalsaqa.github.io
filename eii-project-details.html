Energy Intensity Index Predictor

A machine learning model designed to forecast Energy Intensity Index values for industrial energy optimization. The hybrid ensemble combines neural networks and Random Forest to improve prediction accuracy and reliability.

Role
ML Developer
Domain
Energy Systems
Stack
Python TensorFlow Scikit-learn

Overview

The Energy Intensity Index (EII) is a key metric for measuring energy efficiency in industrial processes. Accurate EII prediction enables proactive decision-making for energy optimization and cost reduction.

This project developed a hybrid ensemble model that combines the pattern recognition capabilities of neural networks with the robustness of Random Forest algorithms, delivering improved forecasting accuracy over traditional regression methods.

Model Architecture

  • Neural Network Ensemble: 10 feed-forward networks trained per dataset, predictions averaged for ensemble robustness
  • Random Forest Component: Complements neural networks with different inductive bias
  • MSE-based Weighting: Final predictions combine both models weighted by their validation performance

Data Processing

  • MAD Normalization: Median Absolute Deviation scaling for outlier resistance
  • Feature Engineering: Automated feature selection based on correlation analysis
  • Cross-Validation: 5-fold CV ensures model generalizes across different data splits

Model Architecture

Ensemble architecture combining neural networks and Random Forest

Outcome

Results

The EII Predictor demonstrated improved forecasting accuracy over traditional regression methods, enabling proactive decision-making in energy-intensive industries. The model supports real-time prediction for operational energy management.

Applications include:

  • Automated energy cost assessments
  • Real-time prediction and analytics dashboards
  • Sustainability metric optimization
