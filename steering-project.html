Formula SAE Steering System Development

Enhancing Driver-Vehicle Interaction through Advanced Embedded Systems

Project Overview

As part of the Queen's Formula SAE team, I contributed to the development of an advanced steering system designed to improve driver feedback and control. This project involved the integration of a custom-designed steering wheel PCB, a TFT display for real-time data visualization, and robust communication protocols.

Key Responsibilities

  • Hardware Integration: Collaborated on the design and implementation of the steering wheel PCB, ensuring seamless integration with vehicle systems.
  • Display Implementation: Utilized the FT800-FT813 graphics controller to manage the TFT display, providing drivers with critical information such as speed, gear position, and system alerts.
  • Communication Protocols: Developed and tested CAN Bus communication protocols to facilitate reliable data exchange between the steering system and other vehicle subsystems.

Technical Highlights

  • Embedded Systems Programming: Wrote and optimized firmware for the steering wheel's microcontroller, focusing on real-time performance and reliability.
  • Graphics Programming: Created custom graphics and interfaces for the TFT display, enhancing user experience and readability under various conditions.
  • System Testing: Conducted comprehensive testing, including hardware-in-the-loop simulations, to validate system performance and robustness.

Project Outcomes

The successful implementation of the steering system resulted in improved driver interaction and control, contributing to the overall performance and safety of the Formula SAE vehicle. The project also provided valuable insights into embedded systems design, real-time data processing, and automotive communication protocols.
