Executive Summary

The Formula SAE project focuses on developing an advanced steering system to enhance driver feedback and control. Utilizing embedded systems and a custom-designed steering wheel PCB, the project emphasizes real-time data visualization and robust communication protocols for seamless integration.

Project Overview

The steering system includes a custom PCB integrated with a TFT display, enabling real-time data visualization for speed, gear position, and system alerts. Collaboration across mechanical, electrical, and software domains ensured a robust and reliable design.

Key Responsibilities

  • Hardware Design: Designed and integrated the custom steering wheel PCB with other subsystems.
  • Real-Time Data Display: Developed user interfaces for the TFT display using the FT800-FT813 graphics controller.
  • Communication Protocols: Implemented and tested CAN Bus protocols for reliable system communication.

Technical Highlights

  • Embedded Systems: Developed and optimized firmware for real-time performance.
  • Graphics Programming: Created dynamic and interactive interfaces for enhanced readability.
  • System Testing: Conducted hardware-in-the-loop simulations to validate performance.

Project Outcomes

The steering system achieved its goal of improving driver interaction and performance while meeting all reliability and safety standards for Formula SAE. This project showcased innovative embedded systems design and effective cross-disciplinary collaboration.
